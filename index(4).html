# Inversiones Bea

Sistema Profesional de Gestión de Préstamos y Cobranza.

## Tecnologías

- React
- Node.js
- Express
- PostgreSQL
- Prisma
- TypeScript
- TailwindCSS

## Ejecutar PostgreSQL

```bash
docker compose up -d
```

## Ejecutar el proyecto

```bash
pnpm install

pnpm dev
```
